# Zakat Calculator for WooCommerce

Advanced Zakat calculator with WooCommerce payment integration.

## Features

- ✅ Comprehensive Zakat calculation
- ✅ Multiple asset types (cash, gold, silver, investments, etc.)
- ✅ Nisab threshold calculation
- ✅ WooCommerce cart integration
- ✅ Multi-currency support
- ✅ Multi-language ready
- ✅ Responsive design
- ✅ Vue.js 3 powered interface

## Installation

1. Upload plugin folder to `/wp-content/plugins/`
2. Activate the plugin through WordPress admin
3. Configure settings under Settings > Zakat Calculator
4. Create a WooCommerce product for Zakat donations
5. Add shortcode `[zakat_calculator]` to any page

## Shortcode

```
[zakat_calculator]
```

## Requirements

- WordPress 5.8+
- PHP 7.4+
- WooCommerce 5.0+

## Configuration

1. Go to Settings > Zakat Calculator
2. Set current Gold price per gram
3. Set current Silver price per gram
4. Select Nisab calculation method (Gold or Silver)
5. Set WooCommerce Product ID for Zakat donations
